Alembic
=======

Alembic is a database migration tool that is tightly integrated with
SQLAlchemy. The models generated by OpenAlchemy are standard SQLAlchemy
models which means that they work with Alembic.

   `Alembic OpenAlchemy example <https://github.com/jdkandersson/OpenAlchemy/tree/documentation/15-example-docs/examples/alembic>`_
      Example of integrating OpenAlchemy with Alembic.
